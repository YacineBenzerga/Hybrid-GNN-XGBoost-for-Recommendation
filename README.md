# Can we Improve XGBoost Performance with GraphSAGE Embeddings?

In this notebook we attempt to combine XGBoost powerful non linear decision boundaries with GraphSAGE node embeddings to improve the performance of our model and provide recommendations that are based on both product features and global neighboring graph information.
